# Skincare Clinic Website

Static skincare clinic website built with HTML, CSS, and JavaScript.

## Deploy

This project can be deployed on GitHub Pages, Netlify, or Vercel as a static site.

For GitHub Pages:

1. Push this repository to GitHub.
2. Open the repository on GitHub.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/ (root)` folder.
6. Save. GitHub will publish the site after the Pages build completes.

The entry file is `index.html`.
